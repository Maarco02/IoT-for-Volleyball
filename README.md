# IoT-for-Volleyball
Part of a project related to the design of an IoT platform based on the exploitation of wearable sensor, multimedia data, wireless communication and AI processing to analyze the movement of athletes playing volleyball.

## Struttura della Repository
* **Classification/**: Contiene la tabella comparativa dei dataset analizzati (formato .csv).
* **Literature/**: Elenco dei paper di riferimento e dei link alle sorgenti dati.

- [x] Raccolta iniziale dataset (Kaggle/Zenodo)
- [ ] Analisi critica frequenze e posizionamento
- [ ] Implementazione algoritmi di classificazione gesti
